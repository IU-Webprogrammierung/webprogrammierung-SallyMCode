# Webprogrammierung – Portfolio

**Projekt:** Web-Programmierung (IU Internationale Hochschule)

Dieses Projekt ist ein persönliches Portfolio, das im Rahmen des Studienfachs Webprogrammierung entwickelt wurde. Es dient dazu, einen kompakten Eindruck von mir als Person, meinen Projekten und meinen Fähigkeiten zu vermitteln.

Das Portfolio umfasst insgesamt sieben responsive Seiten:
- **Home** (Willkommensseite)
- **Lebenslauf**
- **Blog**
- **Auf Reisen**
- **Fotografien**
- **GitHub-Projekte**
- **Browsergame** (Tic Tac Toe)

## Tech-Stack
- **HTML5** (Semantische Struktur & ARIA-Barrierefreiheit)
- **CSS3** (Flexbox, Grid & Media Queries für responsives Design)
- **JavaScript** (Spiellogik und interaktive Elemente)

## Features
- **Fokus auf Barrierefreiheit:** Optimiert für Screenreader (ARIA-Rollen) und vollständige Tastaturbedienbarkeit (inkl. Skip-Link)
- **Zweisprachigkeit:** Vollständige Sprachauswahl zwischen Deutsch und Englisch
- **Darkmode:** Auswählbares, kontrastreiches Farbschema für angenehme Lesbarkeit
- **Responsives Design:** Optimierte Darstellung vom Smartphone bis zum Desktop
- **Interaktives Browsergame:** Ein vollständig in JavaScript implementiertes Tic Tac Toe

## Projekt starten
Da das Projekt ohne schweres Backend auskommt, kann es einfach repliziert werden:
1. Repository klonen: `git clone https://github.com/SallyMCode/webprogrammierung-SallyMCode.git`
2. Die `index.html` direkt in einem beliebigen Browser öffnen.
