Weather Dashboard

A weather dashboard in a vintage style, retrieving current weather data through its own Node.js backend.

Features:

  • Real-time weather data: Fetching temperature, wind speed and humidity via OpenWeatherMap API.
  • Secure backend: Utilizing a Node.js/Express proxy server to protect API keys.
  • Responsive: Optimized for desktop and mobile devices.

Tech-Stack:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express
  • API: OpenWeatherMap

The project is available on GitHub: Weather Dashboard on GitHub (opens in new tab)
